# Slovakia Data Map — H Heuristics

Interactive data map of Slovakia featuring 2,927 municipalities across 8 NUTS3 regions. Explore economy, demographics, education, and infrastructure indicators.

## Live Site

Visit the live site at: https://hunterhghs.github.io/Slovakia-Data-Map/

## Features

- **2,927 municipalities** at LAU level with population density choropleth
- **8 NUTS3 regions** with GDP, employment, education, and broadband indicators
- **4 thematic categories**: Economy, Demographics, Education & Innovation, Infrastructure
- **Interactive map** with search, tooltips, layer toggle, and category switching
- **Comprehensive data tables** with regional comparisons
- **Professional H Heuristics design** — Cormorant Garamond, Source Sans 3, navy & gold

## Data Sources

- Eurostat GISCO 2024 — administrative boundaries (© EuroGeographics)
- Eurostat API — GDP, employment, population, education, broadband statistics
- ŠÚSR Census 2021 — municipality-level demographics

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- D3.js v7 — interactive map rendering
- H Heuristics Brand Design System
- GitHub Pages deployment

## Local Development

```bash
python3 -m http.server 8080
```
